Este sistema WEB é um E-Commerce que simula uma loja de jogos de vídeo-games.

Tem como objetivo aprender a integrar o backend utilizando node.js e expressJs

com o frontend usando Angular Material

Este sistema tem funcionalidade de adicionar,editar e criar produtos para o administrador

e os clientes podem fazer compras, utilizando funcionalidade do carrinho.
